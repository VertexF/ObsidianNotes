# The operating systems process list

The operating system has to have data structures that store the processor list. The **process control block** or **task list** the name data structure that holds all processes. 

When the process is paused with it's blocked or paused it's deloaded from the registers and into memory somewhere, the process is being scheduled it loads back into the CPU registers to be ran again. This process is called **context switch.**
