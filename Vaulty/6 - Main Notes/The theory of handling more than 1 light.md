# The theory of handling more than 1 light

Illuminations is linear so you can easily add more than 1 light source. You compute the light seperately and sum them all to obtain the contribution of light.

You can randomly sample lights with a good algorithm using only some of the light sources for each shaded point in the scene because this light linearity, with ray tracing.
