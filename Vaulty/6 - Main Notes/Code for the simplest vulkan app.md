# Code for the simplest vulkan app

The simplest vulkan you can make can be found in "Examples/SimplestTriangleRenderer". This renderer has all the set up display a triangle with the minimum amount of vulkan code. It does support window resizing. 

The vertices are hardcoded into the shader.
