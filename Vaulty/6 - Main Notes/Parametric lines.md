# Parametric lines

Given twos points $p_{1}$ and $p_{2}$ you get this function for a line.
$$L(t) = (1 - t)p_{1} + tp_{2}$$
For rays this would this would be defined over $[0, \infty)$ and for a line it would be $(-\infty, \infty)$ for a line we can easily re-write into this formula.
$$L(t) = p + t\vec{v}$$Were $\vec{v}$ is just $p_{2} - p_{1}$ it's often the case that $\vec{v}$ is a normalised vector so $t$ corresponds to the length of $\vec{v}$.
